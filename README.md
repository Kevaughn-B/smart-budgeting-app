# smart-budgeting-app
A Full-Stack web app that helps users track spending, categorize expenses, set budgets, and recieve bill reminders with FastAPI, React, PostgreSQL, Docker, and CI/CD following industry SDLC practices.


# Highlights
## Sprint 1 Highlights

- Designed and implemented a FastAPI backend using clean architecture principles
- Established SQLAlchemy database layer with session management
- Created first domain model (User) and RESTful API endpoint
- Implemented health monitoring endpoint for service observability
- Followed Git feature-branch workflow and sprint-based development
